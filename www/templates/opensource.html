How to contribute

This an open source App


What does it mean?

It means that anybody can download, use, modify or redistribute the source code of this app in order to improve it or do a derivate work. The only concern is that the content data is protected under the Open Game License and if you want to include it, you should check the License option for more details.


How can you help?

Sure! There's always room for suggestions, enhancements, fixes and more.

If you want to get the source code and contribute your work, check the repository and start coding.

If you want to report a bug or suggest an enhancement, please report them here or email me.


Why should you care?

Well... Is up to you.

But this App is distributed for free and without ads, just for the love to the game. It was made using the game's own open system and open technologies. Each bit of contribution helps a lot, so don't be afraid to give your own!
